📋 Contexte de la procédure

Installation d'une imprimante partagée sur un poste utilisateur Windows via le serveur d'impression de l'entreprise. Le pilote est automatiquement déployé depuis le serveur.

🎯 Avantage serveur d'impression : Le pilote est automatiquement téléchargé depuis le serveur - aucune installation manuelle requise de la part de l'utilisateur.

⚙️ Procédure d'installation utilisateur

1

Ouvrir les paramètres d'imprimantes

Accéder aux paramètres des imprimantes via le menu Démarrer de Windows.

⚙️ Panneau de configuration > Périphériques et imprimantes

Méthode rapide :

Windows + R → taper "control printers" → Entrée

2

Ajouter une imprimante réseau

Cliquer sur "Ajouter une imprimante" et sélectionner l'option d'imprimante réseau.

➕ Ajout d'une imprimante réseau

Navigation :

"Ajouter une imprimante" → "L'imprimante que je veux n'est pas dans la liste"

3

Sélection par nom

Choisir "Sélectionner une imprimante par nom" et entrer le chemin du serveur.

Ajouter une imprimante

Option : Sélectionner une imprimante par nom

Chemin : \\SRV-PRINT-01\IMP_Service_Admin

💡 Format : Utiliser le format \\NOM_SERVEUR\NOM_PARTAGE
4

Téléchargement automatique du pilote

Windows contacte le serveur d'impression et télécharge automatiquement le pilote.

🔧 Installation automatique depuis le serveur
🎯 Installation silencieuse : L'utilisateur n'a rien à faire - le pilote approprié est automatiquement sélectionné et installé depuis le serveur.
5

Confirmation d'installation

Windows confirme la réussite de l'installation de l'imprimante.

Installation de l'imprimante

Statut : Installation réussie

Imprimante : IMP_Service_Admin sur SRV-PRINT-01

Pilote : HP Universal Printing PCL 6 (installé depuis le serveur)

Option recommandée :

Cocher "Définir comme imprimante par défaut" si nécessaire

6

Test d'impression

Effectuer un test d'impression pour valider le fonctionnement.

🖨️ Test d'impression depuis le poste utilisateur

Méthode de test :

Clique droit sur l'imprimante → "Propriétés de l'imprimante" → "Imprimer une page de test"

7

Vérification finale

Vérifier que l'imprimante apparaît correctement dans la liste des périphériques.

Élément Valeur attendue Statut
Imprimante visible IMP_Service_Admin OK
Statut Prêt OK
Test d'impression Page de test réussie OK
Pilote installé Depuis serveur OK

🔑 Bénéfices de l'installation via serveur

✅ Simplicité : Installation en 3 clics pour l'utilisateur
✅ Standardisation : Pilote unique et approuvé par l'IT
✅ Maintenance : Mises à jour centralisées sur le serveur
✅ Sécurité : Pas de téléchargement de sources externes
✅ Support : Résolution rapide des problèmes

🛠️ Résolution des problèmes courants

Problème : "Impossible de se connecter à l'imprimante"
Solution : Vérifier la connectivité réseau vers le serveur (ping SRV-PRINT-01)
Problème : "Pilote non disponible"
Solution : Contacter l'IT pour vérifier les pilotes sur le serveur
Problème : "Accès refusé"
Solution : Vérifier les permissions sur le partage d'imprimante
