PPE 2 – Architecture Haute Disponibilité

Projet BTS SIO SISR

1. Contexte

L’entreprise CICAR est une agence de location de véhicules implantée aux Îles Canaries. Elle possède plusieurs agences reliées à un siège principal. Afin d’améliorer la fiabilité de son site de réservation, l’entreprise souhaite mettre en place une infrastructure réseau sécurisée et hautement disponible.

2. Objectifs

  • Assurer la haute disponibilité du site web
  • Mettre en place une architecture 3-tiers
  • Sécuriser les flux réseau
  • Garantir la continuité de service

3. Architecture

  • Pare-feu Stormshield
  • DMZ avec HAProxy
  • Deux serveurs Apache
  • VMware ESXi
  • Active Directory

Schéma réseau

Internet | [Stormshield Firewall] | DMZ (192.168.64.0/24) | [HAProxy] | ----------------------------- | | WEB-01 WEB-02 192.168.54.15 192.168.54.16 | | LAN (192.168.54.0/24) | [AD / DNS / DHCP]

4. Mise en œuvre

  1. Création des machines virtuelles
  2. Configuration réseau (LAN / DMZ)
  3. Mise en place du pare-feu
  4. Installation HAProxy
